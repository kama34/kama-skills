# Outline — Генератор структуры презентаций

Автор: Камышников Дмитрий — больше можно глянуть в канале: https://t.me/typicalinnopolisyanin

## Что это?

Outline — мета-скилл для Claude Code, который создает **пайплайны агентов** для итеративной проработки структуры презентаций. Вместо фиксированного шаблона он проектирует специализированных агентов (генератор, ревьюер, корректировщик и т.д.), заточенных под конкретный тип презентации и аудиторию.

## Как это работает

Каждый "шаблон" — это набор агентов, которые работают циклически:

1. **Генератор** создает начальную структуру
2. **Ревьюер** оценивает её от лица аудитории (например, как инвестор, ученик, коллега)
3. **Корректировщик** исправляет структуру на основе замечаний
4. Цикл повторяется, пока ревьюер не одобрит или не будет достигнут лимит итераций
5. **Опциональные агенты** запускаются после основного цикла (например, генератор вопросов Q&A)

## Команды

```
/outline <тема>                               Создать структуру (автоподбор шаблона)
/outline --template <имя> <тема>              Использовать конкретный шаблон
/outline --format <slidev|universal|custom>   Переопределить формат выхода
/outline --create-template                    Создать новый шаблон (интерактивный визард)
/outline --learn=N [шаблон]                   Обучение: N тестовых прогонов, критик корректирует промпты
/outline --help                               Показать справку
```

## Форматы выхода

- **slidev** (по умолчанию) — `## Slide N: Title` + буллеты, совместим с `/slidev`
- **universal** — секции, тезисы, заметки спикера — не привязан к инструменту
- **custom** — формат определяется самим шаблоном

## Шаблоны

Шаблоны хранятся как директории с промптами агентов:

```
<имя-шаблона>/
  ├── template.md       # Метаданные: имя, описание, ключевые слова, формат
  ├── pipeline.md       # Порядок запуска агентов, циклы, критерии остановки
  └── agents/
      ├── generator.md
      ├── reviewer.md
      ├── fixer.md
      └── ...
```

Где хранятся:
- **Локально:** `.outline-templates/<имя>/` (для конкретного проекта)
- **Глобально:** `~/.claude/outline-templates/<имя>/` (доступны отовсюду)

## Создание шаблонов

Запустите `/outline --create-template` — Claude Code задаст несколько вопросов (имя, описание, аудитория, формат), а затем **самостоятельно спроектирует пайплайн агентов**: решит сколько агентов нужно, напишет их промпты и настроит порядок выполнения.

## Обучение агентов

Запустите `/outline --learn=N [шаблон]` для улучшения промптов агентов. Скилл генерирует N тестовых тем, прогоняет пайплайн, и агент-критик анализирует результаты и предлагает улучшения промптов.

## Для контрибьюторов

Этот скилл открыт для экспериментов и улучшений. Идеи:
- Создать новые шаблоны для конкретных типов презентаций
- Улучшить дефолтных агентов
- Добавить новые форматы выхода
- Улучшить критика в `--learn`

Если улучшите что-то — буду рад, если поделитесь результатом!
