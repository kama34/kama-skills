# Improvements from Learn Iteration 3

## Applied Changes

### Change 1: Adjacent slide structural fingerprint check
- **File**: SKILL.md, Step 4.5 deck-level constraints
- **Type**: add_rule
- **Description**: Different archetype names can produce identical visual layouts. Added fingerprint check for adjacent slides.

### Change 2: Section divider visual weight on bg-accent
- **File**: SKILL.md, Step 4.5
- **Type**: add_rule
- **Description**: Section dividers on teal bg-accent looked empty. Added requirement for prominent decorative element (circle, rule, ghost number, arc) occupying 20%+ of slide.

### Change 3: Icon shape by semantic purpose not position
- **File**: SKILL.md, Step 5 Icon Container Selection
- **Type**: modify_rule
- **Description**: Mechanical circle→rounded→ghost rotation was predictable AI-tell. Changed to semantic assignment: circle=primary, rounded=secondary, ghost=tertiary, with cross-slide variation.

### Change 4: Decorative motifs opacity on bg-alt
- **File**: SKILL.md, Step 5 Decorative Layer
- **Type**: add_rule
- **Description**: Teal decoratives invisible on warm grey bg-alt. Added 1.5x opacity multiplier for bg-alt slides.
