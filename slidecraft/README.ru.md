# SlideCraft

[![en](https://img.shields.io/badge/lang-English-blue)](README.md) [![ru](https://img.shields.io/badge/lang-Русский-green)](README.ru.md)

Двухслойный генератор презентаций: AI-сгенерированные фоны + текстовый оверлей на Slidev.

## Использование

```
/slidecraft <аутлайн или файл>
/slidecraft --help
```
